# Project Description
This project analyzes current Clash Royale trends using data such as popular decks, card usage rates and win percentages. We want to understand how the meta is evovolved over time and got to where it is today. This data and analysis will allow us to identify which strategies are currently most effective in the game. By extracting and analyzing data from the Clash Royale API, we can and will have the current trends of the game to provide insights into the competitive landscape of Clash Royale.

# Project Outline/Plan
- Define the specific Clash Royale trends we want to study
- Collect data on deck usage, card popularity, and win rates
- Store the data in a clean and organized format
- Analyze the data for patterns and changes in the meta
- Create visualizations such as bar charts, trend graphs, and rankings
- Summarize the findings in the final project submission

# Data Collection and Storage Plan (written by Author #1)
Using the Clash Royale API, I will request data from currrent leaderboards, either in specific regions or globally.
Since the data requested will be in format .json, I will extract and clean up the data and store it in .txt format with only important or relevant information for the analysis.
